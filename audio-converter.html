Note: the language/voice used for each cell comes from that column's dropdown at the top of the sheet (same as the main page) — set the source column's language there before converting. Browsers can't hand a script the raw audio bytes speechSynthesis produces, so this records the sound the browser actually plays: when you click "Convert Range" your browser will ask you to share this tab — tick "Share tab audio" in that dialog, or nothing will be captured. Files download as .webm (plays in any browser, VLC, and most phones) since browsers can't natively encode true .mp3 client-side — say the word if you'd like a true-.mp3 version added. Each output cell gets a 🎵 (play) and ⬇ (re-download) icon once its file is ready, and the sheet moves down one row at a time: read → capture → save → next row.
